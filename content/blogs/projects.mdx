## Installation Guide

To begin using MDX, install the required packages.

```bash
pnpm add mdx-bundler
pnpm add gray-matter
pnpm add react
```

Create a simple MDX file:

```md
# Hello World

This is an MDX document.
```

Then render it using your preferred MDX renderer.

## Folder Structure

A common project structure might look like:

```txt
app/
├─ page.tsx
├─ posts/
│  ├─ hello-world.mdx
│  ├─ nextjs-guide.mdx
│  └─ react-hooks.mdx
components/
├─ mdx-components.tsx
core/
├─ parser.ts
├─ renderer.tsx
└─ toc.ts
```

This structure separates rendering logic from content and makes scaling easier.
